# "for" লুপে কি কি মান দেখা যাবে?

প্রতিটা লুপে যা দেখা যাবে তা লিখুন। এরপর প্রদত্ত উত্তরের সাথে তুলনা করুন।

দুটি লুপই কি একই মান `alert` করে?

১. পোস্টফিক্স ফর্ম:

    ```js
    for (let i = 0; i < 5; i++) alert( i );
    ```
২. প্রিফিক্স ফর্ম:

    ```js
    for (let i = 0; i < 5; ++i) alert( i );
    ```
