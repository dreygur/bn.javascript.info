# while লুপটি কোন মানগুলি দেখাবে?

লুপের প্রতিটি পুনরাবৃত্তির জন্য, এটি কি মান আউটপুট দেয় তা লিখুন এবং উত্তরের সাথে তুলনা করুন।

দুটি লুপই কি একই মান `alert`করে?

1. শুরুতে `++i`:

    ```js
    let i = 0;
    while (++i < 5) alert( i );
    ```
2. শেষে `i++`

    ```js
    let i = 0;
    while (i++ < 5) alert( i );
    ```
